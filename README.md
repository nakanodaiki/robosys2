# robosys2

数をカウントするプログラムです。
roscoreを立ち上げcount.pyとtwice.pyをはしらせます。
その後rostopic echo /count_upと打つと数が1づつ増えながら表示され、
rostopic echo /twiceと打つと偶数が表示されます。
